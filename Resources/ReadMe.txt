The Java Project components are grouped in the folder "LibraryContact". The project needs to be imported to application and executed.

The application requires MySql to be hosted in the machine to run. Initial set up of database and data is provided alongwith.

The JAR file for the connecting to mysql needs to be added as External JAR reference to the project. The JAR file is also provided herewith.

The dates needs to be entered of format YYYY-MM-DD. In order to notify the users the front end application displays 1900-01-01 as default date.
